常见问题

📋 如何上传体检报告？

打开「档案」页面，点击右下角 + 按钮，选择 PDF 文件导入即可。支持大多数体检机构出具的 PDF 报告。

🤖 如何使用 AI 分析功能？

首先在「设置 → AI 模型配置」中添加 API 密钥（支持 DeepSeek、智谱 GLM、Kimi 等）。配置完成后，在报告详情页点击「开始分析」或使用「AI」对话功能。

👨‍👩‍👧‍👦 如何管理家庭成员？

在「家庭」页面中创建家庭组并添加成员。添加后，在上传报告、录入病例时可选择「为谁录入」来关联家庭成员。

🔒 我的数据安全吗？

所有健康数据均存储在您的设备本地，不会上传到任何服务器。API 密钥使用 iOS Keychain 加密存储。详情请查看我们的隐私政策。

⚠️ AI 分析结果准确吗？

AI 生成的内容仅供参考，可能存在误差。如有身体不适，请及时前往医院就诊，以医生的专业诊断为准。
