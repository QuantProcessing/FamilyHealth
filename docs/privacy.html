隐私政策

家庭健康AI版  |  最后更新：2025年3月24日

核心承诺：家庭健康AI版 的所有健康数据均存储在您的设备本地。我们不运营任何服务器，不收集、不存储您的任何个人信息。
⚠️ 重要声明：本应用为个人开源项目，AI 生成的内容仅供参考，不构成任何医疗诊断或治疗建议。如有身体不适，请务必及时就医。

1. 我们收集的信息

我们不收集任何个人信息。具体来说：

  • 您的体检报告、病例记录等健康数据仅保存在您的 iPhone 本地存储中
  • 您的家庭成员信息仅保存在设备本地
  • 我们不使用任何分析工具或追踪 SDK
  • 我们不收集设备标识符、位置信息或使用行为数据
  • 我们不运营任何后端服务器

2. AI 服务与数据传输

本应用使用第三方大语言模型 API 提供 AI 健康信息整理功能。当您使用 AI 功能时，相关文本数据会发送至第三方 AI 服务商。

2.1 内置免费模型

应用内置了一个免费 AI 模型（由 SiliconFlow 提供）。使用该模型时：

  • 您的对话文本和体检报告摘要会发送至 SiliconFlow 的 API 服务器
  • 数据传输通过 HTTPS 加密
  • 请参阅 SiliconFlow 隐私政策 了解其数据处理方式

2.2 用户自配置模型

您可以自行配置其他 AI 服务（如 DeepSeek、智谱 GLM 等）。使用这些服务时：

  • 对话文本和报告摘要会发送至您所选择的 API 服务商
  • API 密钥由您自行提供，存储在设备的 iOS Keychain 中（加密存储）
  • 我们不会截获、存储或转发您与 AI 服务商之间的通信
  • 请自行了解所选服务商的隐私政策和数据处理规范

2.3 发送的数据类型

数据类型发送场景接收方
对话文本AI 对话用户配置的 AI API 服务商
体检报告文本摘要AI 分析报告用户配置的 AI API 服务商
病例记录摘要AI 对话中引用用户配置的 AI API 服务商

注意：原始 PDF 文件和图片不会发送至 AI 服务商，仅发送提取的文本内容。

3. Apple 健康数据

本应用可选择读取 Apple 健康中的数据（步数、心率、血氧、睡眠等）。这些数据：

  • 仅在您明确授权后读取
  • 仅在设备本地使用和展示
  • 当用于 AI 对话时，相关数据摘要可能发送至 AI 服务商
  • 不会用于广告、数据挖掘或出售给第三方
  • 您可以随时在系统设置中撤销权限

4. 数据存储与安全

  • 所有数据使用 Apple SwiftData 框架存储在设备沙盒中
  • API 密钥使用 iOS Keychain 加密存储
  • 数据随设备的锁屏密码 / Face ID / Touch ID 保护
  • 与 AI 服务商的通信使用 HTTPS 加密传输
  • 卸载应用将永久删除所有本地数据
  • 您可以在应用内「设置 → 数据管理」中手动删除所有数据

5. 数据共享

我们不会将您的数据出售、出租或共享给任何第三方用于广告或营销目的。

数据仅在以下情况下传输至第三方：

  • 当您主动使用 AI 功能时，对话内容发送至您选择的 AI 服务商
  • 除此之外，没有任何数据离开您的设备

6. 第三方服务

本应用不包含任何第三方广告、分析或追踪 SDK。应用中唯一的网络通信是 AI API 调用。

7. 儿童隐私

本应用不针对 13 岁以下儿童设计，也不会有意收集儿童的个人信息。

8. 用户权利

您拥有以下权利：

  • 数据删除：您可以随时在应用内删除所有数据，或卸载应用以永久删除
  • 撤销授权：您可以随时在系统设置中撤销 Apple 健康访问权限
  • 选择 AI 服务：您可以自由选择或停止使用 AI 功能

9. 隐私政策变更

如果我们对本隐私政策进行重大变更，将通过应用更新通知您。建议您定期查看本页面。
