隐私政策

家庭健康AI版  |  最后更新：2025年3月18日

核心承诺：家庭健康AI版 的所有健康数据均存储在您的设备本地，我们不会收集、上传或共享您的任何个人健康信息。

1. 我们收集的信息

我们不收集任何个人信息。具体来说：

  • 您的体检报告、病例记录等健康数据仅保存在您的 iPhone 本地存储中
  • 您的家庭成员信息仅保存在设备本地
  • 我们不使用任何分析工具或追踪 SDK
  • 我们不收集设备标识符、位置信息或使用行为数据

2. AI 服务

本应用支持接入第三方大语言模型 API（如 DeepSeek、智谱 GLM、OpenAI 等）进行健康数据分析。请注意：

  • AI API 密钥由您自行配置，存储在设备的 Keychain 中
  • 与 AI 模型的对话内容会发送至您所配置的 API 服务商
  • 请参阅相应 AI 服务商的隐私政策了解其数据处理方式
  • 我们不会经由我们的服务器转发您的任何数据

3. Apple 健康

本应用可选择读取 Apple 健康中的数据（步数、心率、血氧等）。这些数据：

  • 仅在您明确授权后读取
  • 仅在设备本地使用，不会上传到任何服务器
  • 您可以随时在系统设置中撤销权限

4. 数据存储与安全

  • 所有数据使用 Apple SwiftData 框架存储在设备沙盒中
  • API 密钥使用 iOS Keychain 加密存储
  • 数据随设备的锁屏密码 / Face ID / Touch ID 保护
  • 卸载应用将永久删除所有本地数据

5. 第三方服务

本应用不包含任何第三方广告、分析或追踪 SDK。唯一的外部通信是您自行配置的 AI API 调用。

6. 儿童隐私

本应用不针对 13 岁以下儿童设计，也不会有意收集儿童的个人信息。

7. 隐私政策变更

如果我们对本隐私政策进行重大变更，将通过应用更新通知您。建议您定期查看本页面。
